# 💰 Smart Expense Tracker

A modern Expense Tracker application developed using **C++**, **HTML**, **CSS**, and **JavaScript**. The project helps users manage their daily expenses through a clean dashboard, secure authentication, expense management, and financial statistics.

---

## 📌 Features

### 👤 User Authentication
- User Registration
- User Login
- Logout
- Session Management using Local Storage

### 💵 Expense Management
- Add Expense
- Edit Expense
- Delete Expense
- View Expense History
- Search Expenses
- Sort Expenses

### 📊 Dashboard
- Total Expense
- Monthly Expense
- Highest Expense
- Lowest Expense
- Average Expense
- Total Expense Records

### 📈 Reports
- Export Expense Report as CSV
- Statistics Modal
- Monthly Summary

### 🎨 User Interface
- Modern Responsive UI
- Dark Mode
- Sidebar Navigation
- Dashboard Cards
- Smooth Scrolling
- Animated Statistics

---

# 🛠 Technologies Used

## Backend
- C++

- File Handling
- REST API

## Frontend
- HTML5
- CSS3
